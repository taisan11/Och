### 開発方法
フォークします
node.jsとpnpmの
環境を用意します
開発環境にクローンして以下のコマンドを入力します
あとD1instanceを生成し、wrangler.tomlに入力package.json、を変更しといてください
```
pnpm i
pnpm dbg
pnpm dbm
pnpm dev
```
D1を利用している関係でD1の機能を使おうとするとなんも起きないので
D1の機能を使いたい場合は以下のコマンドを使用してください
```
pnpm build
pnpm preview
```

まだ開発段階なので本番環境での使用はやめてください
